## Key Interface Components

### 1. Header Section
- **Logo & Brand**: NEXUS AI TRADING logo and text
- **Market Status Ticker**: Shows live market status and ticker symbols
- **Account Information**: Portfolio value, credits balance
- **Broker Selection**: Dropdown for selecting trading broker (NinjaTrader, Alpaca, InteractiveBrokers, Tradovate)

### 2. Tab Navigation
The application has 5 main tabs:
- **🎯 Live Trading**: Main trading dashboard for active trading
- **📊 Dashboard**: Performance metrics and account overview
- **🤖 Strategy Hub**: AI trading strategy management
- **🛒 Strategy Store**: Marketplace for purchasing/downloading strategies
- **📈 Analytics**: Detailed performance analysis and reporting

### 3. Status Bar
- **Mode Indicator**: Shows the current operating mode (DEMO, LIVE, etc.)
- **ML Service Status**: Connection status to the ML/AI backend services
