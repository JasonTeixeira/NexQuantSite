## 🆘 **Troubleshooting**

### **Common Issues**

#### **"Python not found"**
- Install Python 3.8+ from python.org
- Make sure Python is added to PATH

#### **"Module not found" errors**
- Run: `pip install -r requirements.txt`
- Or use the launcher which auto-installs packages

#### **App won't start**
- Check Python version: `python --version`
- Try: `python desktop_app.py` directly
- Check the console for error messages

#### **Charts not displaying**
- Install matplotlib: `pip install matplotlib`
- Check if tkinter is available: `python -c "import tkinter"`

### **Performance Issues**
- Close other applications to free memory
- Use smaller date ranges for faster backtests
- Enable batch processing in config
